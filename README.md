# Jimeng Agent

即梦抖音竖屏短视频脚本生成器 — 根据主题生成 9:16 竖屏全维度分镜提示词 + 音效建议。

## 使用方式

在 Reasonix Code 中通过 `/skill jimeng-script` 触发，或直接描述视频主题。

### 交互流程

1. 提供视频主题/需求 → Agent 输出大纲（含创作说明 + 钩子设计）
2. 确认大纲 → Agent 展开完整分镜提示词（可直接导入即梦 seedance 2.0）

### 示例

输入：「AI口语陪练App，目标用户备考雅思的大学生」

输出：4 段分镜（25 秒），每段含全维度画面提示词 + 音效建议
